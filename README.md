~/.zsh.d
====

## Installation

```bash
% cd
% git clone https://github.com/shinyaishida/.zsh.d.git
% .zsh.d/setup.sh
```

## Usage

Edit ```.zsh.d/common``` to specify snippets to be loaded from ```.zshrc```. Snippets are stores in files named ```zsh_*```.
